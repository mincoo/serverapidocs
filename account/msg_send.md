##单发消息

####1.基本信息
- 接口地址：`{API_URL}/msg/send`
- 请求方式：POST

####2.参数信息  

| 参数名    | 必填      | 说明      |
| -------   |:-------:  |-----------|
| uid       | 是        | 用户ID    |
| token     | 是        | 用户令牌  |
| sign      | 是        | 数据校验码|
| nid       | 是        | 接收用户ID|
| gid       | 是        | 接收者群组ID|
| title     | 是        | 消息标题  |
| content   | 是        | 消息内容  |
| url       | 否        | 消息url   |

####3.报文响应

```
{
	"status":1,
	"msg":"success",
	"data":
}
```

|键      |类型  |说明  |
|--------|------|------|
|status  |int   |发送结果<br>1 成功<br>0 失败|
|msg     |String|结果信息描述|
