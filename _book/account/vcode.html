获取验证码

1.基本信息

  • 接口地址：{API_URL}/user/vcode
  • 请求方式：POST

2.参数信息

参数名 必填 说明
phone 是 用户手机号
type 是 获取短信验证码类型
1 注册账户
2 找回密码
3 修改手机
sign 是 数据校验码

3.报文响应

{
    "status":1,
    "msg":"success",
    "data":
}
键 类型 说明
status int 发送结果
1 成功
0 失败
msg String 结果信息描述
